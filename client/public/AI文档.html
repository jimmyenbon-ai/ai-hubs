Enbon AI Hub 平台操作指南

访问地址：http://192.168.20.40:3005（公司局域网内任意电脑浏览器打开即可）

自由创作 使用步骤

  1. 左侧选择「AI 图片生成」→ 点击「自由创作」

平台 · AI 工作流（核心功能）

AI 工作流是本平台的核心特色：通过可视化拖拽编排，实现从需求分析 → 知识库搜索 → 提示词生成 → 图片/视频/音乐生成 → 结果输出的全自动流程。

什么是工作流？

就像一个"自动化 AI 生产线"：你输入一个想法（如"为FC Pro做一张海外社媒海报"），系统自动帮你完成：分析需求 → 搜索产品素材 → 生成提示词 → 出图 → 汇总输出。无需逐个环节手动操作。

支持多人同时使用，各自运行互不干扰。

Enbon · AI 视觉生产力培训（企划部内部）

目标：让平面 / 文案 / 3D 同事在 AI 时代更快产出、更稳更高级，并形成统一的工作流与资产沉淀。

一句话原则：你负责“表达目标与审美判断”，AI 负责“把目标翻译成专业提示词/脚本/分镜”。

建议阅读顺序：先看「平台指南」了解工具操作 → 再看「核心思路」→「提示词公式」→「图片/视频/文案实操」→「资产库与规范」→「排错」。

1. 核心思路与训练目标

AI 并不是“替你工作”，而是“帮你把想法变成可执行的指令”。真正提升效率的关键是：你能不能把脑海的想法说清楚，并让 AI 输出专业、精准、详细、可复用的提示词/脚本。

传统方式（低效率）

  • 凭感觉写提示词：抽象、短、缺少约束
  • 生成结果随机：风格漂移、质感不稳
  • 返工靠“再试几次”：不可控、不可复用

推荐方式（高效率）

  • 先说“要表达什么”（信息/情绪/定位/人群）
  • 再让 AI “拆解成提示词结构”（可复制）
  • 用负面约束 + 校验标准，稳定出片

本次培训的核心训练：训练大家形成一句话定义能力：“这张图/这段视频要表达什么，而不是它长什么样”

例子： 问ChatGPT/豆包或者其他AI:
我想要把这张白底产品图，渲染成一张拥有DJI质感的高清质感产品渲染图，背景深邃、加一些光带、不要有任何变形，请你给我一段完整、详细、精准的提示词，要求产品结构、细节不要有任何变形。

那么AI就会根据你的核心需求'DJI质感、渲染质感'，以及'产品结构、细节不要有任何变形'的约束，生成一段完整、详细、精准的提示词。

参考图

把想法、参考图给AI：
我想把我们Enbon的LED显示屏产品，让nanobanana生成我给你的图这种质感，高级、渲染质感，包括背后加一些光环、光带之类的装饰，高级感，我这边能够提供我们LED显示屏产品的各种角度图，你能给我一段精准、详细的提示词吗？要求不要改变我们产品任何细节和结构。

AI的回复：

基于上传的 Enbon LED 显示屏真实产品图片生成一张顶级工业级产品渲染视觉。

                    【产品结构与细节规则（绝对强约束）】
                    这是一个真实存在、已量产的工业产品。
                    必须 100% 保留原始产品的：
                    - 外形轮廓
                    - 箱体比例
                    - 结构布局
                    - 模组排列
                    - 边框厚度
                    - 螺丝位置
                    - 接口、把手、锁扣等所有工业细节
                    严禁任何形式的：
                    - 重设计
                    - 结构变形
                    - 简化
                    - 拉伸
                    - 压缩
                    - 增减零件
                    - 改变材质形态
                    
                    产品本体只能在【真实材质基础上提升渲染质感】，而不是改变设计。
                    
                    【整体风格】
                    风格参考：高端旗舰手机 / 顶级工业产品发布视觉
                    关键词：高级、克制、精致、工程美学、理性、高端、可信赖
                    整体感觉：昂贵、专业、国际一线品牌水准

                    - 使用电影级产品布光
                    - 边缘有柔和而精准的轮廓光（rim light）
                    - 金属部分呈现细腻、真实、干净的高端工业反射

                    - 明暗层次清晰但不过曝
                    - 对比度控制高级、克制

                    背景为纯粹的高端抽象空间，不出现任何真实场景。

                    - 柔和的光环（halo）
                    - 抽象的光带、弧形光轨
                    - 类似能量流、光波、光晕的视觉元素
                    
                    规则：
                    - 所有光效必须处于产品后方或远景
                    - 不得接触、覆盖或干扰产品主体
                    - 光效为低对比、柔和、渐变式
                    - 不出现硬边、杂乱纹理、图案或颗粒噪点
                    
                    整体空间应有明显纵深感，背景与产品分离清晰。

                    - 产品可轻微悬浮或立于极简暗色地面
                    - 地面为深色、平滑、无纹理
                    - 仅允许非常轻微的接触阴影或微弱倒影
                    - 不出现明确地平线

                    - 超高清
                    - 真实感渲染

                    禁止霓虹色污染

要素 说明（写给 AI 的信息） 示例（ENbon 常用）

2. AI 工具选择与分工

我们用 AI 的原则：免费优先、能识图优先、按任务选工具。没有谷歌账号时，国产工具优先推荐：豆包、DeepSeek。

2.1 分工：谁负责“想”，谁负责“做”

  • “想” —— 对话型 AI（生成提示词/脚本/结构）
    适合：拆需求、写提示词、改写文案、生成分镜、做对比方案
    常用：ChatGPT / Gemini / Grok / DeepSeek / 豆包
  • “做” —— 生产型 AI（生成图片/视频/音频）
    适合：视觉素材、场景图、海报背景、动效视频、配乐
    常用：NanoBanana（图像）/ 可灵、即梦（视频）/ Suno（音乐）

2.2 工具选择建议（快速决策表）

任务 首选 备选 注意事项
提示词/脚本 能识图的对话 AI DeepSeek / 豆包 要给“参考图 + 约束 + 输出格式（分段/要点/可复制）”
产品质感图 NanoBanana 其他图生图工具 强调“不是产品设计工具”，必须保结构比例不变
海报背景动效 可灵 / 即梦 剪辑软件加动效 “文字/Logo/产品”尽量锁定；动背景做高级变化
配乐（无歌词） Suno 素材库 明确：B端、高级、克制、无主旋律过于洗脑

3. 统一视觉生产流程（总览）

ENbon 推荐标准流程（SOP）：

  1. 收集素材：产品实拍/渲染、Logo、字体、既有海报、参考风格图。
  2. 一句话定义：这次要表达什么（目标/受众/风格/约束）。
  3. 对话AI生成提示词：拆成“强约束 + 风格表达 + 负面约束 + 输出规格”。
  4. NanoBanana 出图：多次生成（3-5次），只选“结构正确 + 质感高级 + 不像AI”。
  5. 视频AI：用首尾帧或锁定画面做动效（背景动、文字分层动）。
  6. 剪辑/转场/节奏：统一节奏（4-6秒一段），每段只表达一个点。
  7. 资产库沉淀：提示词、源文件、版本、成品统一存档，可复用。

3.1 角色分工

  • 企划（你）：定义目标与策略、统一审美方向与判断标准、沉淀模板与资产库
  • 平面/3D：出图与质量把控、结构正确/质感高级、二次排版与终稿输出
  • 文案：卖点语言与节奏、CTA/标题/短句、分镜解说词/字幕

4. 提示词方法论（核心）

提示词不是“描述图片”，而是“下达指令”。要做到：可复制、可迭代、可验收。

4.1 提示词四段式（强烈推荐）

【A 强约束】必须保持不变的东西（产品结构/比例/Logo/文字/位置等）
【B 目标表达】这张图/这段视频要表达什么（品牌调性/情绪/卖点）
【C 风格与摄影语言】镜头、光线、材质、背景氛围、构图留白、色调
【D 负面约束】明确禁止：变形、重绘Logo、乱码文字、廉价光效、复杂纹理等

4.2 提示词=“可执行规格书”，不是作文

不推荐（太虚）：“做一张高级科技感的产品图，背景好看一点。”

推荐（可执行）：“产品结构100%不变；主体占画面40%-55%，左/右留白；背景为低对比渐变与远景抽象光带；灯光为克制的工业棚拍高光；禁止纹理、禁止复杂元素、禁止重绘Logo与任何文字变化。”

4.3 对话AI提问模板

6. 视频生产：可灵 / 即梦 工作流（首尾帧与背景动效）

视频想要高级：不是堆特效，而是“克制的镜头 + 稳定的层级 + 清晰的节奏”。

6.1 推荐制作方式 A：首尾帧动画（稳定、好控）

  • 每段 4–6 秒；每段只表达一个点（例如：轻 / 快装 / 防护 / 散热）
  • 首帧：爆炸图/特效图/重点细节；尾帧：完整箱体图（或反过来）
  • 产品可静止，重点用：镜头轻推、景深变化、背景光影流动

6.2 推荐制作方式 B：海报“文字不动、背景动”

  1. 准备：海报 PNG（文字/Logo/产品必须清晰），另备“无文字纯背景版”（可选）
  2. 提示词重点：锁定文字与Logo，背景做微动（渐变流动、柔光、光带、粒子极少）
  3. 输出：竖屏 9:16（TikTok/Reels）或 1:1（IG/FB），时长 6–10 秒
  4. 剪辑：统一转场（淡入/擦拭/推拉），统一字体与字幕样式

9. 常见翻车与排查清单（最实用）

9.1 结构被改/产品变形

解决：把“产品冻结”放到提示词最顶端，并加重复强调（100% unchanged）；背景元素必须“远景抽象、低对比”。

9.2 文字乱码/Logo 变成文字

解决：提示词写明 Logo 必须使用“提供的图形标识”；视频尽量“锁定文字层”。

9.3 画面廉价、像“炫光PPT”

解决口诀：少、远、软、暗

  • 少：减少元素数量；每屏只留 1 个视觉中心
  • 远：装饰必须远景抽象
  • 软：软阴影、软渐变、软高光
  • 暗：对比克制，避免高饱和霓虹

【负面】禁止复杂背景、禁止实体装饰物靠近产品、禁止纹理图案、禁止霓虹炫光、禁止任何变形。

11. 练习任务与验收标准

11.1 练习任务（每人至少做 1 套）

  2. 让对话AI产出提示词：用“四段式”输出3个版本（工程/品牌/传播）。
  3. 用NanoBanana出图：至少生成3次，选1张最佳，并写出“为什么选它”。
  4. 做一个6秒背景动效：锁定文字/Logo不动，只动背景；导出竖屏或方图。
  5. 归档：把提示词+参考图+结果+结论存进资产库，并按命名规范命名。

11.2 验收标准

  • 合格（必须）：产品结构比例细节无变化、Logo/文字无改动、无乱码、画面干净、留白合理、层级清晰、提示词可复用
