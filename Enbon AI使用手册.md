## 1. 系统概述

### 1.1 平台简介

Enbon AI 智能设计工作台是一站式 AI 内容生成平台，专为 LED 显示屏外贸营销场景打造。平台集成了**图片生成、视频生成、音乐生成、文案撰写、知识库管理**等核心能力，并通过**可视化工作流**将多个 AI 能力串联成自动化内容生产线。

### 1.2 核心能力一览

| 能力模块 | 支持模型 | 主要用途 |
|---------|---------|---------|
| AI 图片生成 | GPT-Image 2 / Nano Banana 系列 | 产品海报、社媒素材、广告创意 |
| AI 视频生成 | Seedance 2.0 / 1.5 Pro 等 | 产品短视频、品牌宣传片 |
| AI 音乐生成 | Suno V3~V5 | 背景音乐、品牌配乐 |
| AI 文案生成 | DeepSeek / GPT-4o | 营销文案、产品介绍、邮件 |
| AI 工作流 | 可视化编排 | 自动化多步骤内容生产 |
| 知识库 | 文档 + 图片存储 | 品牌资产统一管理 |

### 1.3 访问方式

局域网内任意电脑浏览器访问：

```
http://192.168.20.40:3005
```

> 服务部署在局域网服务器上，所有生成的内容（图片、视频）会永久保存在服务器本地，不会丢失。

## 2. 快速入门

### 2.1 界面布局

打开平台后，界面分为三个区域：

```
┌──────────┬──────────────────────────────┐
│          │                              │
│  左侧    │        主工作区               │
│  导航栏  │                              │
│          │                              │
│  ·图片   │    （根据选择的模块           │
│  ·视频   │     显示不同操作界面）         │
│  ·音乐   │                              │
│  ·工作流 │                              │
│  ·模板库 │                              │
│  ·设置   │                              │
│          │                              │
└──────────┴──────────────────────────────┘
```

- **左侧导航栏**：切换不同 AI 功能模块
- **主工作区**：当前模块的操作界面
- **顶部栏**：显示当前页面标题、主题切换、积分余额

### 2.2 主题切换

平台支持 4 种主题配色，点击顶部右侧按钮切换：

- **深色** — 护眼暗色模式（默认推荐）
- **灰色** — 中性灰色调
- **浅色** — 明亮白底
- **蓝色** — 商务蓝调

### 2.3 积分系统

每次生成内容会消耗积分，不同模型消耗不同：

| 操作 | 消耗积分 |
|------|---------|
| GPT-Image 2 生成 | 2 分/张 |
| GPT-Image 2 VIP | 5 分/张 |
| Nano Banana 系列 | 1~3 分/张 |
| Seedance 2.0 视频 | 10 分/条 |
| Seedance 2.0 Fast 视频 | 5 分/条 |
| Suno 音乐 | 2~8 分/首 |

> 积分余额显示在顶部栏右侧，低于 10 分会变红提醒。积分由管理员统一充值。

## 3. AI 图片生成

### 3.1 自由创作模式

这是最常用的图片生成方式，适合单张创意输出。

#### 操作步骤

**第 1 步：选择模型**

在左侧模型卡片中点击选择一个：

- **GPT-Image 2**：通用画质，速度快，适合日常素材
- **GPT-Image 2 VIP**：高级画质，支持 2K/4K 分辨率，适合印刷级输出
- **Nano Banana Pro**：性价比高，风格多样
- **Nano Banana 2**：最新版本，4K 支持

**第 2 步：输入提示词**

在提示词输入框中用中文或英文描述你想要的图片内容。一个好的提示词包含：

```
[主体] + [场景/背景] + [风格] + [画质词]

示例：
一位商务女性在LED显示屏前做产品演示，现代办公室背景，
专业商务风格，柔和灯光，8K超高清，细节丰富
```

**快捷标签**：输入框下方提供了"风格标签"和"画质标签"，点击即可插入。

**提示词快捷库**：点击"提示词快捷库"可插入已保存的常用提示词片段。

**第 3 步（可选）：上传参考图**

点击上传区域或拖拽图片到虚线框内，可上传 1~9 张参考图。上传后在提示词中使用 `@1`、`@2` 引用它们。

**第 4 步：选择参数**

- **画面比例**：1:1（方形）/ 16:9（横屏）/ 9:16（竖屏）/ 4:3 / 3:4 等
- **分辨率**：1K（标准）/ 2K（高清）/ 4K（超清，VIP 模型支持）
- **画质**：自动 / 低 / 中 / 高（仅 GPT 模型支持）

**第 5 步：生成**

点击"生成图片"按钮或按 `Ctrl + Enter` 快捷键。

#### 结果管理

生成结果展示在右侧"我的作品"面板中：

- **列表/网格切换**：切换展示模式
- **收藏**：点击星标收藏优秀作品
- **下载**：保存图片到本地
- **查看详情**：查看完整提示词和参数
- **搜索筛选**：按关键词、日期范围搜索历史作品

### 3.2 提示词撰写技巧

#### 产品展示类

```
LED显示屏产品棚拍，纯白背景，专业产品摄影，
正面角度，屏幕点亮显示彩色画面，8K商业摄影品质
```

#### 场景应用类

```
大型户外LED广告屏安装在现代化商业广场，
夜晚场景，屏幕上播放品牌广告，人群驻足观看，
电影级光影，超广角视角
```

#### 社媒素材类

```
LED透明屏在高端零售橱窗展示，时尚店面背景，
屏幕播放动态产品展示，ins风，明亮自然光
```

## 4. 批量图片生成

当需要一次性生成多张图片（如产品多角度、多配色方案）时使用。

### 4.1 准备文件

支持以下格式的批量输入文件：

**TXT 文件**（推荐新手使用）：
```
蓝色LED租赁屏户外音乐节现场搭建效果
同款屏幕室内会议场景
产品纯白背景棚拍
```

> 每行一个提示词，空行分隔。

**Markdown 文件**：
```markdown
## 户外场景
LED租赁屏户外音乐节现场
LED广告屏商业广场安装效果

## 室内场景
LED会议屏企业会议室
LED透明屏零售橱窗
```

> `##` 标题作为分组名称。

**CSV/Excel 文件**：
| 提示词（必填） | 模型（选填） | 比例（选填） | 分辨率（选填） |
|---------|---------|---------|---------|
| 户外LED屏展示 | nano-banana-2 | 16:9 | 2K |
| 室内会议屏 | gpt-image-2 | 4:3 | 1K |

### 4.2 操作流程

1. 点击左侧"批量生成"
2. **上传文件**：拖拽或选择准备好的文件
3. **预览确认**：检查解析出的提示词列表，可逐条修改或删除
4. **设置默认参数**：选择模型、比例、分辨率（应用于所有条目）
5. **开始生成**：点击按钮，实时查看每张图片的生成进度
6. **下载结果**：支持单张下载或一键打包下载 ZIP

## 5. 一键制作模板

模板功能适合**固定品牌风格、批量套用**的场景，如产品系列图、活动海报等。

### 5.1 创建模板

1. 点击左侧"新建模板"
2. 填写模板信息：
   - **名称**：如"GE-系列产品海报"
   - **图标**：选择标识 emoji
   - **分类**：如产品图、活动海报
   - **提示词模板**：使用 `{变量名}` 作为占位符，如：
     ```
     {产品型号} LED显示屏产品棚拍，{背景风格}风格，
     专业产品摄影，{角度}视角，8K超清
     ```
3. **上传参考图**：固定使用的品牌图、Logo、产品图（最多 5 张）
4. **基础底图**：上传设计底版，AI 只替换特定元素，保留布局和品牌标识
5. **选择模型和参数**
6. 保存模板

### 5.2 使用模板

1. 在左侧导航栏点击已创建的模板
2. 在"填写信息"标签页中输入各变量值
3. 系统自动生成完整提示词并提交
4. 在"生成记录"标签页查看历史结果

> 模板适合高频标准化场景，一次创建，反复使用。

## 6. AI 视频生成

### 6.1 四种生成模式

#### 模式一：文生视频

直接用文字描述生成视频，无需上传素材。

```
示例提示词：
LED显示屏生产线自动化展示，机械臂精准组装屏幕模组，
明亮工厂环境，流畅运镜，电影级画质
```

#### 模式二：图生视频（首帧）

上传一张图片作为视频起始画面，AI 从这张图开始生成动态视频。

适用场景：产品图片转动态展示视频。

#### 模式三：图生视频（首尾帧）

上传首帧和尾帧两张图片，AI 生成从首帧过渡到尾帧的完整视频。

适用场景：产品前后对比、安装过程演示。

#### 模式四：多模态参考

上传多张图片/视频/音频作为参考素材，在提示词中用 `@1`、`@2` 引用。

适用场景：品牌调性视频、多素材混剪。

### 6.2 参数说明

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 模型 | Seedance 2.0 效果最好 | Seedance 2.0 |
| 分辨率 | 480p/720p/1080p | 1080p（高清） |
| 比例 | 横屏 16:9 / 竖屏 9:16 / 方形 1:1 | 16:9（社媒）/ 9:16（抖音） |
| 时长 | 2~15 秒 | 5 秒 |
| 种子 | 相同种子可复刻相同视频 | -1（随机） |
| 生成音频 | 是否附带背景音 | 开启 |
| 水印 | 是否带水印 | 关闭 |

### 6.3 查看与管理

- 生成中的任务显示实时进度
- 完成后可**在线播放**、**下载 MP4**
- 历史记录支持搜索、日期筛选、分页

## 7. AI 音乐生成

### 7.1 两种创作模式

#### 灵感模式

只需一句话描述音乐风格，AI 自动创作：

```
一首欢快的电子流行音乐，适合科技产品宣传片背景
```

#### 自定义模式

精确控制音乐内容：

- **歌词内容**：输入自定义歌词（中英文均可）
- **风格标签**：如 Pop, Electronic, Upbeat, Orchestral
- **排除标签**：不希望出现的风格元素
- **是否纯音乐**：勾选"纯器乐"生成无歌词版本

### 7.2 模型选择

| 模型 | 特点 |
|------|------|
| V5（Bluejay 推荐） | 最新最强，音质最佳 |
| V4 | 稳定可靠 |
| V3.5 | 生成速度快 |
| V3 | 经典版本 |

### 7.3 高级功能

- **延长模式**：基于已有歌曲继续生成（需提供源歌曲 ID 和起始时间）
- **翻唱模式**：用新的风格覆盖已有歌曲（需提供源歌曲 ID）

## 8. AI 工作流

工作流是本平台最核心的功能，通过**拖拽连线**将多个 AI 节点串联成自动化内容生产线。

### 8.1 工作流概念

一个典型的工作流包含以下节点，按箭头方向依次执行：

```
[输入想法] → [LLM分析] → [知识查询] → [LLM生成] → [图片生成] → [输出]
```

每个节点完成一个特定任务，上游节点的输出自动成为下游节点的输入。

### 8.2 节点类型详解

#### 分析类节点

**LLM 分析** — 智能分析输入内容

输入一段想法/需求，AI 自动提取：
- 核心主题
- 目标受众
- 风格方向
- 关键元素
- 建议搜索的知识库文件夹

#### 知识类节点

**知识查询** — 从知识库检索相关内容

根据上游分析结果，自动在知识库中搜索匹配的品牌资料、产品信息、设计素材等。支持：
- 按文件夹精准搜索
- 自动分析检索到的参考图风格
- 将产品参数、品牌文案注入后续生成

#### 生成类节点

**LLM 生成** — AI 文案/提示词生成

四种输出模式可选：
- **提示词模式**：生成图片生成用的英文提示词
- **文案模式**：生成营销文案、邮件、社媒内容
- **模板模式**：套用自定义模板，使用 `{{变量}}`
- **分析模式**：做通用文本分析

**文案生成** — 专用营销文案生成

四种文案风格：
- 营销风格 — 促销、转化导向
- 社媒风格 — 轻松、话题性
- 产品介绍 — 参数、卖点突出
- 故事风格 — 品牌故事、场景叙事

**图片生成** — 调用 AI 绘画

支持 GPT-Image 2 和 Nano Banana 全系列模型，可选择分辨率和比例。

**视频生成** — 调用 Seedance

自动整合上游的提示词和参考图，生成视频。

**音乐生成** — 调用 Suno

根据风格分析和知识库内容生成背景音乐。

#### 控制类节点

**条件分支** — 根据条件走不同路径

支持两种判断方式：
- **关键词匹配**（快速）：匹配成功走"是"，否则走"否"
- **LLM 智能判断**：适合复杂条件

**循环迭代** — 重复执行指定次数

对循环体内的节点重复执行 N 次（1~100）。

#### 输出节点

**输出** — 汇总所有上游结果

收集文本、图片 URL、视频 URL、音频 URL、分析结果等。

### 8.3 操作指南

#### 创建第一个工作流

**步骤 1**：点击左侧"AI 工作流"进入工作流编辑器

**步骤 2**：从左侧节点列表拖拽节点到画布

建议从最基础的流程开始：

```
LLM 分析 → 知识查询 → LLM 生成 → 图片生成 → 输出
```

**步骤 3**：连接节点

从一个节点的输出圆点拖拽到下一个节点的输入圆点。

**步骤 4**：配置节点

点击节点，右侧弹出属性面板：
- **LLM 节点**：选择 LLM 配置和输出模式
- **图片节点**：选择模型、分辨率、比例
- **知识节点**：选择查询的文件夹

**步骤 5**：输入想法

在底部输入框中输入你的需求，如：

```
帮我们最新的 GE-P3.9 户外租赁屏做一套 Facebook 推广素材，
需要突出产品的防水防尘优势和户外演出场景效果
```

**步骤 6**：点击"执行"按钮

实时观察每个节点的执行进度（绿色=完成，蓝色=运行中，红色=失败）。

#### 画布操作快捷键

| 操作 | 快捷键 |
|------|--------|
| 撤销 | Ctrl + Z |
| 重做 | Ctrl + Y / Ctrl + Shift + Z |
| 复制选中节点 | Ctrl + C |
| 粘贴 | Ctrl + V |
| 删除选中节点 | Backspace / Delete |
| 自动排列 | 工具栏"自动排列"按钮 |

#### 工作流管理

- **保存**：保存为模板，下次直接使用
- **导出/导入**：JSON 格式，可分享给同事
- **历史记录**：查看每次执行的结果和详情

### 8.4 实战案例

#### 案例一：产品社媒素材批量生成

```
[输入产品型号] → [LLM分析] → [知识查询(产品资料)] → [LLM生成(提示词)] → [图片生成] → [输出]
```

**效果**：输入一个产品型号，自动查知识库获取参数和品牌调性，生成英文提示词，输出多张营销图片。

#### 案例二：带文案的产品短视频

```
[输入需求] → [LLM分析] → [知识查询] → [LLM生成(文案)] → [LLM生成(提示词)] → [图片生成] → [视频生成] → [输出]
```

**效果**：从需求到文案到图片到视频，全自动输出完整短视频素材包。

#### 案例三：A/B 测试素材生成

```
[输入想法] → [LLM分析] → [条件分支(风格=A)] → [图片生成A]
                      → [条件分支(风格=B)] → [图片生成B]
                                  ↓
                              [输出汇总]
```

**效果**：同一主题生成两套不同风格的素材，用于 A/B 测试。

## 9. 知识库管理

知识库是工作流的"大脑"，存储品牌资料、产品信息、设计素材，供 AI 工作时检索引用。

### 9.1 文件夹结构

建议按以下结构组织：

```
知识库
├── 产品资料
│   ├── 户外租赁屏系列
│   ├── 室内会议屏系列
│   └── 透明屏系列
├── 品牌素材
│   ├── Logo
│   ├── VI规范
│   └── 品牌文案
├── 设计参考
│   ├── 优秀案例
│   ├── 风格参考
│   └── 竞品素材
├── 公司资料
│   ├── 公司介绍
│   ├── 资质证书
│   └── 工厂照片
└── 其他
```

### 9.2 支持的文件格式

| 格式 | 类型 | 说明 |
|------|------|------|
| .txt / .md | 文本 | 直接读取内容 |
| .pdf | 文档 | 自动提取文字 |
| .docx / .doc | Word | 自动提取文字 |
| .jpg / .png / .webp / .gif | 图片 | 保存为参考图，附带描述 |
| .svg | 矢量图 | 保存为参考图 |

### 9.3 操作指南

#### 上传文件

1. 进入知识库（工作流界面左侧工具栏 → 知识库图标）
2. 选择目标文件夹
3. 拖拽文件到上传区域
4. 点击"上传"按钮

> 支持批量上传，单次最多 10 个文件，单个最大 20MB。

#### 搜索知识

- 在搜索框输入关键词，实时过滤
- 支持搜索标题、文件名、文件夹路径、内容全文

#### 批量管理

1. 勾选多条知识条目
2. 使用"移动到文件夹"批量整理
3. 使用"批量删除"清理过期内容

### 9.4 最佳实践

- **定期更新**：新产品发布后及时上传资料
- **命名规范**：文件名使用"品牌_型号_类型"格式，如 `Enbon_GE-P3.9_产品参数.pdf`
- **控制文件大小**：图片建议压缩到 2MB 以下，大文件影响搜索速度
- **精选素材**：只上传高质量参考图，避免低质量素材干扰 AI 判断

## 12. 常见问题

### Q1：生成图片一直转圈不结束？

检查 API Key 是否正确配置（设置 → 对应 API Key），检查积分余额是否充足。

### Q2：图片生成后过几天变成裂开/404？

**已修复**。现在生成的图片会自动永久保存在服务器本地。如果是之前生成的老图片出现此问题，是因为第三方 API 的临时链接过期，新生成的不会再有此问题。

### Q3：工作流执行到某一步失败了？

点击失败的节点查看错误信息。常见原因：
- API Key 未配置或过期
- 提示词触发内容审核
- 积分不足
- 参考图链接不可访问

### Q4：如何查看积分余额？

顶部栏右侧实时显示积分余额。点击可手动刷新。

### Q5：上传的参考图点击不起作用？

确认图片上传成功（有缩略图预览）。在提示词中使用 `@1`、`@2` 格式引用，注意是英文的 @ 符号。

### Q6：视频生成需要多长时间？

文生视频通常 30~90 秒，图生视频稍长。任务提交后可以关闭页面，系统后台会自动轮询并更新结果。

### Q7：可以同时在多台电脑上使用吗？

可以。平台部署在局域网服务器上，多台电脑浏览器同时访问 `http://192.168.20.40:3005` 即可。

### Q8：如何备份工作流和知识库？

所有数据保存在服务器 `server/cache/` 和 `server/local_storage/` 目录下。建议定期备份这两个目录。
